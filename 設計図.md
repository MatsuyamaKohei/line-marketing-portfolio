> **フレームワーク**: 7R × CoT/RISE

## 入力情報

| 項目 | 内容 |
|------|------|
| **業種/商材** | 美容室（カット・カラー・トリートメント・ヘッドスパ） |
| **ターゲット層** | 25〜40歳の女性／髪のダメージ・くせ毛・白髪の悩み／「通うたびにキレイになる自分」を求めている |
| **最終CVR** | LINE経由での来店予約（初回クーポン利用） |

## 1. 【全体コンセプト】アカウントの方向性とペルソナ設定

### アカウントコンセプト

> **「あなた専属のビューティアドバイザーが、LINEの中に。」**

- ホットペッパーのような"価格比較の場"ではなく、**「私だけの美容パートナー」**というポジションを確立する。
- 一方的な宣伝ではなく、**ヘアケア知識の提供 → 信頼構築 → 自然な来店動機の醸成**という流れで設計。
- 行動経済学の「返報性の原理」を活用し、先に価値（知識・特典）を提供してから予約へ導く。

### ペルソナ設定

| 項目 | 詳細 |
|------|------|
| **名前** | 佐藤 あかり（仮） |
| **年齢** | 32歳 |
| **職業** | 事務職（正社員） |
| **家族構成** | 夫＋子ども1人（3歳） |
| **髪の悩み** | カラーの退色が早い、毛先のパサつき、白髪がちらほら |
| **美容室選びの基準** | 価格より「仕上がりの満足度」と「居心地の良さ」 |
| **情報収集手段** | Instagram、LINE、口コミサイト |
| **心理的願望** | 忙しい毎日でも"キレイでいる自分"を維持したい |
| **来店頻度の理想** | 2ヶ月に1回 → 1.5ヶ月に1回へ短縮させたい |

### アカウントのトーン＆マナー

- **口調**: 丁寧だけど親しみやすい（「〜ですよ！」「〜してみてくださいね✨」）
- **世界観**: 清潔感・上品さ・ナチュラル感（ホワイト×ベージュ×ゴールドの配色イメージ）
- **絵文字の使用**: ✨💇‍♀️🌿💆‍♀️ など上品系に統一
- **配信頻度**: ステップ配信後は月2〜3回（ブロック率を抑制）

## 2. 【リッチメニュー設計】UXを最適化する項目案

### レイアウト：2段×3列（6枠構成）

```
┌─────────────────┬─────────────────┬─────────────────┐
│                 │                 │                 │
│   📋 ご予約      │   💇‍♀️ メニュー    │   🎁 クーポン     │
│                 │                 │                 │
├─────────────────┼─────────────────┼─────────────────┤
│                 │                 │                 │
│   📖 ヘアケア     │   📍 アクセス     │   💬 お問い合わせ  │
│      コラム      │                 │                 │
└─────────────────┴─────────────────┴─────────────────┘
```

### 各メニュー詳細

| メニュー | タップ先 | 目的 | 設計意図 |
|---------|---------|------|---------|
| **📋 ご予約** | 予約フォーム（Lステップ or 外部URL） | 来店予約の導線を最短化 | 最も目立つ左上に配置。タップ→即予約の体験 |
| **💇‍♀️ メニュー＆料金** | カルーセルメッセージ | メニューと価格の確認 | 写真付きカルーセルで視覚的に訴求、各メニューに「予約する」ボタンを配置 |
| **🎁 会員限定クーポン** | クーポン画面 | リピート促進 | 「LINE限定」の希少性を活用。月替わりクーポンで再来店動機を創出 |
| **📖 ヘアケアコラム** | 記事リスト（リッチメッセージ） | 信頼構築・ブロック防止 | 有益な情報提供でアカウントの存在価値を維持 |
| **📍 アクセス** | Google Maps連携 or 店舗情報ページ | 来店ハードルの低減 | 地図・住所・営業時間・駐車場情報をワンタップで確認 |
| **💬 お問い合わせ** | 1:1チャットモード切替 | 不安解消・個別対応 | 「気軽に聞ける」安心感でブロック率を低減 |

### タブ切替設計（Lステップ活用時）

| タブ | 対象者 | 表示内容 |
|------|--------|---------|
| **デフォルト** | 全友だち | 上記6枠構成 |
| **予約済みユーザー** | 来店予約済の友だち | 「予約確認」「来店前の準備」「アクセス」に差し替え |
| **VIPリピーター** | 3回以上来店 | 「VIP限定メニュー」「紹介特典」を追加表示 |

## 3. 【ステップ配信シナリオ】全7通

### シナリオ全体設計

```
友だち追加
  ↓ 即時
あいさつメッセージ（クーポン配布＋アンケート誘導）
  ↓ 1日後
[STEP 1] 信頼構築（自己紹介＋想い）
  ↓ 2日後
[STEP 2] 教育①（悩みへの共感＋知識提供）
  ↓ 2日後
[STEP 3] 教育②（ビフォーアフター事例）
  ↓ 2日後
[STEP 4] 教育③（ホームケアのコツ）
  ↓ 3日後
[STEP 5] セールス①（初回限定オファー）
  ↓ 2日後
[STEP 6] セールス②（お客様の声＋限定感）
  ↓ 3日後
[STEP 7] クロージング（期限リマインド＋最終CTA）
```

---

### STEP 1｜信頼構築 ― 「はじめまして、〇〇サロンです」
- **配信タイミング**: 友だち追加の1日後
- **目的**: サロンの人柄・こだわりを伝え、「このサロンは信頼できる」と感じてもらう
- **内容**:
  - オーナー or スタイリストの自己紹介（顔写真付き）
  - 「なぜこのサロンを始めたのか」ストーリー
  - 「お客様の髪を本気で考えるサロンです」という想い
- **CTA**: 「次回、"髪がパサつく本当の原因"をお届けしますね✨」（次回予告で離脱防止）
- **心理トリガー**: 返報性（価値を先に提供）、ストーリーテリング

---

### STEP 2｜教育①「髪がパサつく"本当の原因"、知っていますか？」
- **配信タイミング**: STEP 1の2日後
- **目的**: 悩みへの共感 → 正しい知識の提供 → 「プロに任せたい」という気持ちの醸成
- **内容**:
  - 「市販シャンプーと美容室専売品の違い」をわかりやすく解説
  - 「ダメージの原因TOP3」を図解で紹介
  - 「でも、正しいケアをすれば髪は生まれ変わります」と希望を提示
- **CTA**: 「明後日、実際のお客様のビフォーアフターをお見せしますね💇‍♀️」
- **心理トリガー**: 認知的不協和（「今のケアで本当に合ってる？」）

---

### STEP 3｜教育②「ここまで変わる！お客様のビフォーアフター」
- **配信タイミング**: STEP 2の2日後
- **目的**: 視覚的な変化でサロンの技術力を証明し、来店意欲を高める
- **内容**:
  - 3名分のビフォーアフター写真（カルーセル形式）
  - 各お客様の悩みと施術内容を簡潔に紹介
  - 「お客様の声」を1〜2件掲載
- **CTA**: 「次回は"おうちでできる簡単ヘアケア術"をプレゼント🎁」
- **心理トリガー**: 社会的証明（他の人もやっている）、バンドワゴン効果

---

### STEP 4｜教育③「プロが教える！おうちヘアケア3つのコツ」
- **配信タイミング**: STEP 3の2日後
- **目的**: 無料で有益なノウハウを提供 → 返報性で信頼残高を最大化
- **内容**:
  - ① 正しいシャンプーの仕方（動画 or イラスト付き）
  - ② ドライヤーの当て方で変わるツヤ感
  - ③ 寝る前の1分ケアで翌朝が変わる
- **CTA**: 「次のメッセージで、LINE限定の"特別なご案内"をお届けします✨」（期待感の醸成）
- **心理トリガー**: 返報性の原理（「こんなに教えてくれるなんて…」）

---

### STEP 5｜セールス①「あなただけの"初回限定"特別プラン」
- **配信タイミング**: STEP 4の3日後
- **目的**: ここまでの教育で高まった信頼を、具体的な来店行動へ転換する
- **内容**:
  - 「LINE友だち限定 初回カット＋カラー ○○%OFF」
  - 通常価格との比較を明記（アンカリング効果）
  - 「＋ヘッドスパ15分を無料プレゼント🎁」（オーバーデリバリー）
  - 有効期限：登録日から30日間
- **CTA**: 「▼ 今すぐ予約する ▼」ボタンを目立つ位置に配置
- **心理トリガー**: 希少性（LINE限定）、アンカリング（価格比較）、損失回避

---

### STEP 6｜セールス②「迷っているあなたへ — お客様のリアルな声」
- **配信タイミング**: STEP 5の2日後
- **目的**: 予約を迷っている層の背中を押す
- **内容**:
  - 「最初は不安でしたが…」から始まるお客様の声 3件
  - よくある質問（Q&A形式）
    - Q. 初めてでも大丈夫？ → A. カウンセリング重視で安心
    - Q. どのくらい持つ？ → A. ○週間キープの実績
    - Q. 子連れOK？ → A. キッズスペース完備
  - 「まだクーポンは使えます✨ 有効期限は〇月〇日まで」
- **CTA**: 「▼ 予約はこちら ▼」ボタン
- **心理トリガー**: 社会的証明、不安の解消、希少性リマインド

---

### STEP 7｜クロージング「まもなく終了 — 最後のご案内です」
- **配信タイミング**: STEP 6の3日後
- **目的**: 期限切れ前の最終リマインドでコンバージョンを最大化
- **内容**:
  - 「初回限定クーポンの有効期限が近づいています」
  - これまでのステップ配信の価値を振り返りダイジェスト
    - 「髪の正しいケア方法をお伝えしました」
    - 「お客様のリアルな声もご紹介しました」
  - 「あとは、あなたが"なりたい自分"へ一歩踏み出すだけ✨」
- **CTA**: 「▼ クーポンを使って予約する（残り○日）▼」
- **心理トリガー**: 損失回避（「使わないともったいない」）、デッドライン効果

### ▼ メッセージ②（カルーセル or リッチメッセージ）

**ビジュアルカード**: サロン内装 + スタイリスト写真

- **タイトル**: 🌿 あなたの"なりたい"を叶える場所
- **説明文**: カウンセリング重視の施術で、一人ひとりに合ったスタイルをご提案します。
- **ボタン①**: 📋 初回クーポンを見る
- **ボタン②**: 💇‍♀️ メニューを見る

---

### ▼ メッセージ③（アンケート誘導）

**回答選択式メッセージ（クイックリプライ）**

```
💬 あなたに最適な情報をお届けするために
　 カンタンな質問に答えていただけますか？

Q. 今、一番気になる髪のお悩みは？
```

**選択肢**:
- 🔸 カラーの退色・色持ち
- 🔸 パサつき・ダメージ
- 🔸 くせ毛・うねり
- 🔸 白髪が気になる
- 🔸 ボリュームが出ない

> ※ 選択した回答に基づきタグを付与し、以降のセグメント配信に活用。
> 例：「パサつき・ダメージ」を選んだユーザーには、トリートメント推しのシナリオへ分岐。

## 5. 【タグ設計】セグメント配信のためのタグ体系

| カテゴリ | タグ名 | 用途 |
|---------|--------|------|
| **流入経路** | `流入_Instagram` / `流入_HP` / `流入_店頭POP` / `流入_紹介` | 集客チャネル分析 |

## 6. 【KPI設計】効果測定の指標

| 指標 | 目標値 | 測定方法 |
|------|--------|---------|
| **友だち追加数** | 月50人以上 | Lステップ管理画面 |
| **ブロック率** | 15%以下 | Lステップ管理画面 |
| **ステップ配信 完走率** | 60%以上 | STEP 7到達率 |
| **初回クーポン利用率** | 25%以上 | クーポン使用数 ÷ 友だち数 |
| **再来店率（3ヶ月以内）** | 40%以上 | 来店タグの付与状況 |
| **LINE経由 月間予約数** | 20件以上 | 予約フォーム経由の計測 |
