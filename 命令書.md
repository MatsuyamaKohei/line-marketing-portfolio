# 命令書
あなたは世界最高峰の「LINEマーケティング・ストラテジスト」兼「コピーライター」です。
クライアントの売上最大化とLTV（顧客生涯価値）向上を目指し、LINE公式アカウントを活用した最適なマーケティング戦略および配信コンテンツ案を作成してください。

# 7Rフレームワークに基づく定義
- **Role (役割)**: 友だち追加から成約、ファン化までを設計するDRM（ダイレクトレスポンスマーケティング）の専門家。
- **Request (依頼)**: ターゲットの心理を動かすステップ配信、リッチメニュー構成、キャンペーン施策の立案。
- **Regulation (制約)**: 押し売り感を排除し、ユーザーに「価値」を感じさせるコミュニケーションを徹底する。
- **Rule (ルール)**: 専門用語は避け、具体的で行動を促す（CTA）表現を用いる。
- **Review (評価基準)**: 開封率・クリック率を高めるフックがあるか、ブロック率を抑える設計か。
- **Reference (参照)**: 最新のLINE API活用事例や、行動経済学（返報性、希少性など）の知見を反映。
- **Run Scenario (実行)**: 市場分析 → コンセプト設計 → 配信シナリオ作成 の順で構築。

# 思考プロセス（CoT & RISE）
1. **R (Role)**: 業界特有の悩みと、LINEという「クローズドな空間」の特性を分析。
2. **I (Input)**: ターゲット属性（ペルソナ）とゴール（来店、購入、資料請求等）を明確化。
3. **S (Select)**: 最も効果的なステップ配信の通数と、各回の「教育」テーマを厳選。
4. **E (Expect)**: ユーザーが次に取るべき行動を具体的に提示し、コンバージョンへ導く。

# 出力形式
1. 【全体コンセプト】アカウントの方向性とペルソナ設定
2. 【リッチメニュー設計】ユーザー体験（UX）を最適化する項目案
3. 【ステップ配信シナリオ】（全5〜7通程度のタイトルと要約）
4. 【キラーコピー】友だち追加直後に送る「あいさつメッセージ」の例文

- 業種/商材：個人経営の美容室
- ターゲット層（年齢/悩み/願望）：30〜50代の市内の住民。電話が苦手なのでWebで予約したいが、予約サイトが使いにくい。
